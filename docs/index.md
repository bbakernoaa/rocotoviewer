# RocotoViewer Documentation

Welcome to the documentation for RocotoViewer, a powerful viewer for Rocoto workflow management systems.

## Overview

RocotoViewer provides real-time monitoring and visualization of Rocoto-based workflow systems. It offers a rich command-line interface and terminal-based UI for managing and monitoring workflows.

## Features

- Real-time workflow state visualization
- Interactive log viewing and filtering
- Workflow dependency graph display
- Task status monitoring
- Configuration management
- Command-line interface for automation
